Ruby Gems歷險記

Jan 13th, 2013 | Comments

想安裝Cloudfoundry的command line tool,所以就這樣來做

sudo gem install vmc

失敗,提示

懷疑是網絡問題,所以查看gem的sources,運行

gem sources

輸出:

依次把其他sources都刪除,只留 http://rubygems.org/

再次運行:

sudo gem install vmc

成功!
